# 🌀 CAD Modeling of Different Types of Springs

## 📌 Overview
This project focuses on the design and 3D modeling of various types of springs using CAD software. Springs are essential mechanical components used for energy storage, shock absorption, and force control in engineering systems. The objective of this project is to understand spring geometry, design parameters, and their applications through parametric modeling.

## ⚙️ Types of Springs Modeled
- Helical Compression Spring  
- Helical Extension Spring  
- Helical Torsion Spring  

## 🛠️ Tools & Software Used
- SolidWorks / Siemens NX (CAD Modeling)  

## 📂 Project Files
- Individual CAD part files for each spring type  
- Assembly files (if applicable)  

## 🚀 How to Use
1. Open the CAD files in SolidWorks or Siemens NX  
2. Explore parametric dimensions such as pitch, coil diameter, and number of turns  
3. Modify parameters to observe changes in geometry  

## 🎯 Learning Outcomes
- Understanding of different spring types and their applications  
- Hands-on experience in parametric CAD modeling  
- Knowledge of key design parameters like pitch, coil diameter, and turns  
- Improved visualization of mechanical components  

## 🔧 Applications
- Automotive suspension systems  
- Industrial machinery  
- Consumer products (pens, weighing scales)  
- Mechanical linkages and load systems
